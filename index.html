How this console works

This console sends each teacher candidate the MDE notice to select #TEACH as their educator preparation provider in MOECS, then tracks who has finished. Anyone who hasn't completed it gets an automatic email reminder every 48 hours until they upload proof.

Step by step

  1. Add candidates on the Add Candidates tab — type them in one at a time, or upload a roster file (.xlsx/.csv). For files, download the Excel template first so the columns match. Both methods build one list you review before importing.
  2. Import the reviewed list. Re-importing the same Student ID updates contact info and never resets anyone's progress.
  3. Send from the Recipients & Status tab — select people and "Send to selected," or "Send to all pending." Each email merges the candidate's Student ID and attaches the step-by-step guide PDF.
  4. Each candidate gets a personal link. They upload a screenshot of their MOECS Enrollment Information screen showing #TEACH as provider, which marks them Completed and stops reminders.
  5. Follow-ups go out automatically every 48 hours to anyone still incomplete, up to the configured limit.
  6. Edit the wording anytime on the Email Content tab using the merge fields shown there.
  7. Clean up test or duplicate records with the red Delete button on a row, or by selecting rows and clicking Delete selected. Deletes ask for confirmation and cannot be undone.
